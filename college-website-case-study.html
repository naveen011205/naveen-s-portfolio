College Website Redesign

Role: UI/UX Designer

Duration: 2 Weeks

Tools: Figma, FigJam

Platform: Responsive Website

Problem Statement

Many college websites present outdated layouts, inconsistent navigation, and scattered information, making it difficult for students, parents, and faculty to quickly access important details such as admissions, courses, events, and announcements.

The goal was to redesign the college website with a modern, responsive, and user-friendly interface that improves accessibility and information discovery.

Project Goal

Design a responsive college website that enables users to:

  • Access academic information quickly
  • Explore courses and departments
  • View campus news and events
  • Find admission details
  • Contact faculty and departments easily

Target Users

Primary Users

  • Prospective students
  • Current students
  • Parents

Secondary Users

  • Faculty
  • Administrative staff
  • Alumni

User Research

Research involved reviewing educational websites and collecting feedback from students regarding common navigation challenges.

Key Findings

  • Important information was difficult to locate
  • Navigation menus were overcrowded
  • Mobile responsiveness was poor
  • Admission details lacked clarity
  • Event announcements were hard to find

User Persona

Priya – Higher Secondary Student

Age: 18

Goal

Find admission information and course details quickly.

Pain Points

  • Confusing navigation
  • Too many pages
  • Poor mobile experience

Design Process

  1. User Research
  2. Competitor Analysis
  3. Information Architecture
  4. User Flow
  5. Wireframing
  6. High-Fidelity UI Design
  7. Prototype Testing

User Flow

↓ Receive Information

Design Decisions

  • Simplified the navigation bar for quicker access
  • Organized content using clear visual hierarchy
  • Designed mobile-first responsive layouts
  • Used reusable UI components for consistency

Challenges

  • Structuring a large amount of academic information
  • Balancing visual appeal with readability
  • Ensuring responsiveness across different screen sizes

Outcome

The redesigned website provides a modern, accessible, and user-friendly experience that makes it easier for students, parents, and faculty to find information. Improved navigation and responsive layouts enhance usability across desktop and mobile devices.

Key Learnings

  • Strong information architecture improves discoverability
  • Responsive design is essential for educational platforms
  • Clear typography and spacing enhance readability
  • User-centered design creates a more engaging digital experience
